# 🐻 Campus Interactive Map | 校園互動地圖

A simple and fun interactive map to explore the campus.
這是一個簡單且有趣的校園互動地圖。

---

## 🎮 How to Play | 如何遊玩

* **Move | 移動:** Use `W`, `A`, `S`, `D` or the **Arrow Keys**.
  使用鍵盤的 `W`, `A`, `S`, `D` 或方向鍵來移動。
* **Explore | 探索:** Walk to the markers to discover campus locations.
  走到地標位置即可發現地點。
